
Nesta atividade, desenvolvi um sistema simples de conversão de moedas utilizando C#, aplicando conceitos de usabilidade e boas práticas de desenvolvimento.

Avaliação de Usabilidade (Heurísticas de Nielsen)

Durante o desenvolvimento, foram aplicadas heurísticas para melhorar a interação do usuário com o sistema:

*'Visibilidade do status do sistema
  O programa informa claramente cada etapa, solicitando dados ao usuário e mostrando o que deve ser feito.

*Correspondência com o mundo real
  Utilização de linguagem simples e familiar, como “reais” e “dólar”.

*Controle e liberdade do usuário
  O usuário pode inserir valores livremente e reiniciar o programa quando desejar.

*Consistência e padrões
  O sistema mantém um padrão nas mensagens e organização do layout no terminal.

*Reconhecimento em vez de memorização
  Todas as instruções são exibidas na tela, facilitando o uso sem necessidade de lembrar comandos.
